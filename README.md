# 🚀 Portfolio Admin Dashboard

A modern, scalable, and responsive **Portfolio Admin Dashboard** built with **Next.js 14**. This project provides a centralized CMS-like interface for managing portfolio content, projects, blogs, categories, website customization, and other administrative features.

---

## ✨ Features

- 📊 Modern Dashboard
- 📝 Blog Management
- 💼 Portfolio Project Management
- 🗂️ Categories Management
- 📬 Contact Messages
- ⚙️ Website Customization
- 🎨 Clean & Responsive UI
- 🌙 Dark Theme Dashboard
- ⚡ Optimized Performance with Next.js
- 📱 Mobile Friendly
- 🔒 Scalable Component Architecture

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/admin-dashboard-nextjs.git
```

Navigate to the project

```bash
cd admin-dashboard-nextjs
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
